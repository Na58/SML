1. [file.py]
   This file contains functions for:
   (1) read feature data from file.
   (2) apply feature selection.
   (3) normalize feature data.
   The feature data has been pre-processed and saved into csv file. 

2. [neuralNet.py]
   This file contains the implementation of different machine learning method
   including neural network, random forest, logistic regression.
   Besides, the test methods are implemented in this file including auc curve,
   accuracy calculation and loss result.

3. [TestCSV.py]
   This file is used for comparing the test result and ploting probability curve.
   (project auxiliary program)


The main entry is in the neuralNet.py file
Data should be put into data/confidential directory
